Crear referencias relativas de archivos

Este tema trata de resolver el problema de que los scripts dejen de funcionar debido a la rutas y sus cambios de entorno. Necesitamos definir una forma de donde inicia nuestro proyecto y a partir de ah donde construir rutas

Objetivo

Necesitamos encontrar una forma de evitar que nuestro proyecto se rompa cuando movamos de lugar un archivo dentro del proyecto, para esto usaremos Referencias Relativas.

Estando dentro de VSCode

Vamos a aprender como crear una raz a nuestro proyecto. Lo que queremos hacer es encontrar una manera de decirle a Python "Mi proyecto empieza aqu" y a partir de aqu comienzan las siguientes rutas. Hay diferentes formas de hacerlo.

Pyprojroot

import pyprojroot Importamos la biblioteca, despus lo que hacemos es pyprojroot.here() y esto nos genera un PosixPath.

Para acceder a mi carpeta de datos.
pyprojroot.here("data").joinpath('raw')

Esto me ayuda a tener un control sobre el proyecto y evitar que las rutas se daen. Solamente si movemos los datos tendremos que actualizar la ruta.

Pyhere

Realiza una tarea similar a la anterior pero tiene una funcin here() que me indica en donde estoy y tambin me regresa un PosixPath, lo que significa que trabaja con Pathlib. Pero tambin significa que contiene una ruta padre, es decir un paso atras, desde donde le indiques.

#importando
import pyhere
#Llamando a la funcin 
pyhere.here() #me dice donde estoy
# Utilizando resolve
pyhere.here().resolve() #me dice donde estoy y me regresa un PosixPath
# Accediendo a una ruta
pyhere.here().resolve()/"data"/"raw"

Con estas 2 libreras tu puedes crear tus rutas para tus proyectos y mejorar tu flujo de trabajo.

Creando Shorcuts

Es muy comn que tengas que realizar ciertas a tareas de manera repetida, entre ellas podran ser:

  • Acceder a la carpeta de datos.
  • Acceder a la carpeta de datos preprocesados.

Entre ellas podramos encontrarnos lo siguiente:
pyprojroot.here().joinpath("data","raw") invocar mucho este comando, lo que podramos hacer es realizar una funcin.

Queremos hacer una funcin que nos permita acceder de una manera rpida a cualquier directorio. Dentro de la funcin queremos obtener una funcin que sea similar a here() pero con rutas personalizables.

def make_dir_function(dir_name):

    def dir_function(*args):
        #La funcin recibe o no tiene limite de argumentos
        return pyprojroot.here().joinpath(dir_name,*args)
    return dir_function

NOTA: Con la funcin no importa la cantidad de argumentos *args

Procedemos a ejecutar la funcin y nos regresa lo siguiente, con n nmero de elementos

Haciendo uso de la funcin.

Tambin puedo hacer uso de los mtodos que tenan las dems libreras

#Usando la funcin
data_dir("cookiecutter-personal","{{cookiecutter.project_slug}}","notebooks")
# Peguntando si existe 
data_dir("cookiecutter-personal","{{cookiecutter.project_slug}}","notebooks").exists()

As de esta forma hemos creado una funcin que me permitir crear shortcuts para acceder a los directorios.

Si quisiramos hacer un shortcut para la carpeta notebooks, haramos lo siguiente y quedara indicado que la variable o funcin noteboooks_dir es tambin una raz.

#Usando la funcin
notebooks_dir = data_dir("notebooks")

Ademas es ms explicito el uso de estas funciones para los directorios, pues podemos nombrarlas como tienen el nombre los directorios.
