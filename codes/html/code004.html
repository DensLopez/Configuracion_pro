Implementar hooks

En este momento ya sabemos como crear una plantilla personalizada para nuestros proyectos.

Pero en esta clase vamos a profundizar un poco mas en una herramienta que nos permitir avanzar mas con el programa cookicutter. Esta herramienta son los HOOKS

HOOKS

Son cosas que se van a ejecutar antes o despus de generar la plantilla de datos.

Imaginemos que el project_slug en una ocasin sea el nombre de un paquete, as que necesitamos verificar que el nombre de este sea una expresin valida.

O por ejemplo podemos hacer que git se ejecute despus de haber creado la plantilla o el proyecto, con esto evitamos realizar las cosas repetitivas.

Vamos a aprender como se usan los Hooks

Vamos a modificar el archivo cookiecutter.json. As modificamos la segunda linea:

"project_slug": "{{ cookiecutter.project_title.lower().replace(' ','_').replace('-','_') }}",

Es importante saber que los hooks son acciones que se van a ejecutar antes o despus de crear el proyecto, por ende son archivos de configuracin y van afuera de la carpeta project_slug.

As que creamos una carpeta especial para ellos, llamada "hooks"

Es importante saber que los hooks pueden ejecutarse con archivos de tipo python (.py) o con tipo bash (.sh).

Creamos 2 archivos diferentes con python debido a su facilidad de implementacin.

  1. pre_gen_project.py
  2. post_gen_project.py

As cuando cookiecutter haga la plantilla, buscar el contenido de esta plantilla y proceder a leer los archivos, para poder realizar ciertas acciones antes y despus de crear la plantilla. Podemos hacer cosas simples, hasta algo mucho mas complejo.

Modifiquemos los archivos:

pre_gen_project.py

import subprocess

MESSAGE_COLOR = "\x1b[34m"
RESET_ALL = "\x1b[0m"

print(f"{MESSAGE_COLOR}Almost done!")
print(f"Initializing a git repository...{RESET_ALL}")

#inicializar repositorio
subprocess.call(['git', 'init'])
#anadir los archivos al stage
subprocess.call(['git', 'add', '*'])
#Hacer el primer commit
subprocess.call(['git', 'commit', '-m', 'Initial commit'])

print(f"{MESSAGE_COLOR}The beginning of your destiny is defined now! Create and have fun!{RESET_ALL}")
